# 靈鬥之刃

原創 2D 像素格鬥遊戲試玩版，使用 TypeScript、Vite 與 Phaser 3 製作。

## 執行

```bash
npm install
npm run dev
```

開啟 Vite 顯示的本機網址。

驗證：

```bash
npm run build
npm run test:e2e
```

## 第二階段功能

- 4 名原創鬥士：凱．刃、米拉．維爾、布朗特．海爾、尼克絲．索爾
- 2 張場地：月影鐵道、燼火道場
- 輕擊 > 中擊 > 重擊 > 必殺取消鏈
- 遠距角色具備投射物必殺
- 能量滿時可使用爆氣
- 簡單 / 普通 / 困難 AI
- 連擊計數、爆氣 HUD、結果統計

## 第三階段功能

- 每名玩家角色都有 3 戰街機路線
- 訓練模式支援重置、假人行為、無限能量與招式資訊 HUD
- 生成素材 manifest 與預載入場景，方便未來替換正式 sprite/audio
- 原創程式音效：UI、命中、格擋、爆氣、投射物、擊倒與勝利回饋
- Playwright e2e smoke tests

## 操作

玩家 1：

- 移動：`WASD`
- 輕擊 / 中擊 / 重擊：`J` / `K` / `L`
- 必殺：`U`
- 爆氣：`O`
- 格擋：`I`
- 衝刺：按住 `S` 加 `A` 或 `D`

玩家 2：

- 移動：方向鍵
- 輕擊 / 中擊 / 重擊：`1` / `2` / `3`
- 必殺：`4`
- 格擋：`5`
- 爆氣：`6`
- 衝刺：按住下方向鍵加左或右方向鍵

主選單：

- 街機模式：`A` 或 `Enter`
- 訓練模式：`T`
- 對戰 AI：`V`
- 本機雙人：`L`
- 操作說明：`C`

選角：

- 玩家 1 角色：`A` / `D`
- 玩家 2 / 假人角色：左 / 右方向鍵
- 場地：`Q` / `E`
- AI 難度：`Z` / `C`
- 開始：`Enter`

訓練：

- 重置：`R`
- 切換假人行為：`T`

除錯判定框：`H`
返回主選單：`Esc`
